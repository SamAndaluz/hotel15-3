Detailed Features List

Below is the detailed list of Feature for Odoo Pos Booking System

Arrange Booking and Reservation Online

  • The module facilitates Booking Option in Odoo POS.
  • Customers can book and reserve the services directly from the POS.

Define Time Slots for the Bookings

  • You can define time slots for booking plans.

Set Unique Reservation Charges

  • You can set unique appointment charges for each plan and time slot
  • The customers will be charged as per the chosen plan and time slot

Customer can only add same Booking at a time to the cart

  • Same booking can only be added to the cart one at a time

Set the Booking Quantity as you want

  • You can set maximum booking quantity at a time
  • Or, can also set the overall maximum quantity for booking slot too

Set the Validity of a Booking

  • You can set the start and end date of a booking type product.
The Booking Product is Displayed with the Tag to differentiate it from other products
A Booking Popup to select the specific Date on which booking is available and you want to book
Select the available Time slots and Qty and Book/reserve your seat.
In Show Booking Button You will get all the booking records, where you can perform required actions.
You can Start, Reschedule or even can cancel your booking from here.
On starting the booking, A orderline is added further you can procced for Payment.
